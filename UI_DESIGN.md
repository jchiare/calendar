### Typography

```css
/* Font Family */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* Font Sizes */
--text-xs: 0.75rem;    /* 12px - timestamps, badges */
--text-sm: 0.875rem;   /* 14px - secondary text, event cards */
--text-base: 1rem;     /* 16px - body text */
--text-lg: 1.125rem;   /* 18px - section headers */
--text-xl: 1.25rem;    /* 20px - page titles */
--text-2xl: 1.5rem;    /* 24px - main headings */
```

## Animations & Transitions

### Event Drag & Drop
```css
/* Dragging state */
.event-dragging {
  transform: scale(1.02);
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.15);
  opacity: 0.9;
  cursor: grabbing;
}

/* Drop target highlight */
.time-slot-drop-target {
  background: var(--primary-100);
  border: 2px dashed var(--primary-500);
}
```

### View Transitions
```css
/* Smooth view switching */
.calendar-view-enter {
  opacity: 0;
  transform: translateX(20px);
}
.calendar-view-enter-active {
  opacity: 1;
  transform: translateX(0);
  transition: all 200ms ease-out;
}
```

### Modal Animation
```css
/* Modal backdrop */
.modal-backdrop {
  animation: fadeIn 150ms ease-out;
}

/* Modal content */
.modal-content {
  animation: slideUp 200ms ease-out;
}

@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(16px) scale(0.98);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}
```

## Accessibility (a11y)

### Requirements
- All interactive elements have visible focus states
- Color contrast ratio minimum 4.5:1 for text
- Events distinguishable by more than just color (patterns/icons optional)
- Screen reader announcements for calendar navigation
- ARIA labels on all buttons and controls
- Keyboard navigation for all interactions

### Focus States
```css
/* Visible focus ring */
:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}

/* Event focus */
.event-card:focus-visible {
  box-shadow: 0 0 0 3px var(--primary-200);
}
```

### Screen Reader Support
```html
<!-- Calendar grid -->
<div role="grid" aria-label="Calendar for January 2026">
  <div role="row">
    <div role="columnheader">Sunday</div>
    <!-- ... -->
  </div>
  <div role="row">
    <div role="gridcell" aria-label="January 15, 2026, 3 events">
      <!-- events -->
    </div>
  </div>
</div>

<!-- Event announcement -->
<div aria-live="polite">
  Event "Team Standup" created for January 15 at 9:00 AM
</div>
```
